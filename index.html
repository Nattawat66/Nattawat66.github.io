วิธีการใช้งาน

เพื่อที่จะโฮสต์เว็บนี้ใน GitHub Pages, คุณต้องทำตามขั้นตอนดังนี้:

  1. เข้าไปที่ https://github.com และสร้าง repository ชื่อว่า username.github.io
  2. อัปโหลดไฟล์ index.html และไฟล์อื่นๆ ที่เกี่ยวข้องลงใน repository
  3. ไปที่หน้า Settings ของ repository และเปิดใช้งาน GitHub Pages
  4. รอจนกว่า GitHub จะทำการสร้างเว็บไซต์ของคุณให้เสร็จสมบูรณ์
  5. เว็บของคุณจะสามารถเข้าถึงได้ที่ https://username.github.io (แทนที่ username ด้วยชื่อผู้ใช้ GitHub ของคุณ)
